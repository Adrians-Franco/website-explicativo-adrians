HTML — Estruturando Páginas

Início HTML CSS JavaScript React Node.js

O HTML é a base de toda página web. Ele define o conteúdo e a estrutura.

Exemplo: Títulos e Parágrafos

<h1>Olá, mundo!</h1>
<p>Este é o meu primeiro site.</p>

Resultado:

Olá, mundo!

Este é o meu primeiro site.

Links e Imagens

O HTML também nos permite conectar páginas (Links) e adicionar elementos visuais (Imagens).

Links (âncoras)

<a href="https://www.google.com" target="_blank">Ir para o Google</a>

O atributo href define o destino, e target="_blank" abre em uma nova aba.

Imagens

<img src="caminho/para/imagem.jpg" alt="Descrição da Imagem">

O atributo src indica onde a imagem está, e alt é crucial para acessibilidade.

Resultado:

Ir para o Google
