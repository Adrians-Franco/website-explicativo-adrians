React — Criando interfaces com componentes

    Início     HTML     CSS     JavaScript     React     Node.js

React é uma biblioteca JavaScript criada pelo Facebook para construir interfaces.

Com React, você cria componentes reutilizáveis para montar telas inteiras.

Componentes Básicos

Exemplo de Componente Simples:

function Ola() {
  return <h1>Olá, mundo em React!</h1>;
}

Props (Propriedades)

As Props são argumentos que você passa para os componentes. Elas permitem que um componente seja **reutilizável** com dados diferentes.

Exemplo de Componente com Props:

// 1. Definição do Componente
function BoasVindas(props) {
  // Acessa a propriedade 'nome'
  return <h2>Boas-vindas, {props.nome}!</h2>;
}

// 2. Uso do Componente (Passando a Prop)
/* Renderizando...
<BoasVindas nome="Aluno" />
<BoasVindas nome="Professor" />
*/

Conceito:

A Prop **nome** faz com que o mesmo componente exiba:

"Boas-vindas, Aluno!"

"Boas-vindas, Professor!"
