O que é Node.js?

Node.js é um ambiente de execução JavaScript que roda no servidor, não no navegador. Isso permite que você use JavaScript para criar aplicações backend, APIs, servidores web e muito mais.

Node.js é baseado no V8, o motor JavaScript do Google Chrome, e é conhecido por sua performance e capacidade de lidar com muitas conexões simultâneas.

Node.js vs JavaScript no Navegador

Node.js

  • ✓ Acesso ao sistema de arquivos
  • ✓ Criar servidores
  • ✓ Acessar banco de dados
  • ✓ Variáveis de ambiente

Navegador

  • ✓ Manipular DOM
  • ✓ Eventos do usuário
  • ✓ LocalStorage
  • ✓ Geolocalização

Começando com Node.js

Primeiro, instale Node.js do site oficial. Depois, crie um arquivo JavaScript:

// app.js
console.log("Olá, Node.js!");

// Executar com: node app.js

Módulos e NPM

NPM (Node Package Manager) permite instalar e gerenciar bibliotecas:

// Inicializar um projeto
npm init

// Instalar uma biblioteca
npm install express

// Usar a biblioteca

const app = express();

    res.send('Olá, Mundo!');

app.listen(3000, () => {
    console.log('Servidor rodando na porta 3000');

Exemplo: API com Express

const express = require('express');
const app = express();

app.use(express.json());

// Dados de exemplo
let usuarios = [
    { id: 1, nome: 'João' },
    { id: 2, nome: 'Maria' }
];

// GET - Obter todos os usuários
app.get('/usuarios', (req, res) => {
    res.json(usuarios);
});

// GET - Obter um usuário específico
app.get('/usuarios/:id', (req, res) => {
    const usuario = usuarios.find(u => u.id === parseInt(req.params.id));
    if (!usuario) return res.status(404).send('Usuário não encontrado');
    res.json(usuario);
});

// POST - Criar um novo usuário
app.post('/usuarios', (req, res) => {
    const usuario = {
        id: usuarios.length + 1,
        nome: req.body.nome
    };
    usuarios.push(usuario);
    res.status(201).json(usuario);
});

app.listen(3000, () => {
    console.log('API rodando na porta 3000');
});
